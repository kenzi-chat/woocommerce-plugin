# CLAUDE.md — Kenzi WooCommerce Plugin

## Supported Platform Versions

This plugin MUST support both WooCommerce 10.4 and 10.5. Update `WC tested up to` in `kenzi-commerce.php` when new versions are verified.

| Platform | Versions | Demo Environments |
|----------|----------|-------------------|
| WooCommerce | 10.4, 10.5 | `woo104.demo.kenzi.chat` (10.4.3), `woo105.demo.kenzi.chat` (10.5.2) |
| WordPress | 6.7+ | 6.7 (demo) |
| PHP | 8.1+ | 8.3 (demo) |

Both versions are continuously tested via the demo environments. Any change to this plugin must work on both WooCommerce 10.4 and 10.5.
